# 数字图像处理（Digital Image Processing）

使用 **数字图像处理(第四版)**

这个仓库会记录下我读书的笔记。
按照自己对书本中原理的理解，挑选书中的例子使用 Python 语言进行实现。

在一期中使用 python/jupyter 实现了关于

1. 频率域滤波
2. 图像复原与重建
3. 图像分割
4. 特征提取
5. 图像分类

的部分内容。
